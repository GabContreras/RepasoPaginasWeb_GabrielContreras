Preguntas actividad remedial

1. ¿Qué es HTML y cuál es su función principal en el desarrollo web?
Lenguaje de marcado de hipertexto, sirve para realizar la base de las páginas web, sin el diseño (para eso es el CSS).
2. ¿Qué son las etiquetas HTML y cómo se utilizan?
Las etiquetas HTML distinguen cada elemento, por ejemplo:
<H1> para títulos, <body> para el contenido de la página, <a> para enlaces.
3. Proporciona ejemplos de algunas de las etiquetas HTML más comunes.
<br> para saltos de línea, <p> para párrafos, <img> para imágenes.
4. ¿Qué es CSS y cómo se utiliza para dar estilo a una página web?
Es un lenguaje usado para diseñar el aspecto visual de una página web.
5. ¿Cuáles son las propiedades más importantes de CSS?
background-color: Color de fondo.
background-image: Imagen de fondo.
margin: Espaciado externo.
padding: Espaciado interno.
6. Explica el modelo de caja (box model) en CSS.
El box model estructura los elementos de una página como cajas con contenido, padding, border y margin.
7. ¿Qué es el diseño responsivo y cómo se logra con CSS?
Es un diseño adaptable a distintos tamaños de pantalla, logrado con flexbox o media queries.
8. ¿Qué es JavaScript y cómo se integra en el desarrollo web?
Es un lenguaje que agrega interactividad a las páginas web. Se integra con la etiqueta <script>.
9. Explica cómo JavaScript interactúa con el HTML y CSS a través del DOM.
El DOM representa la estructura del documento HTML en forma de árbol, permitiendo modificar elementos en tiempo real.
10. Proporciona ejemplos de cómo JavaScript puede hacer que una página web sea interactiva.
alert("¡Bienvenido!"); Muestra una alerta.

Haz clic en el botón para cambiar este texto.

11. ¿Qué es Git y por qué es importante para el desarrollo web?
Es un sistema de control de versiones que ayuda a guardar y rastrear cambios en el código.
12. Explica cómo Git se usa para el control de versiones en proyectos de desarrollo web.
Git permite volver a versiones anteriores del código. Con git log se puede ver el historial de cambios.
13. ¿Qué es GitHub y cómo facilita la colaboración en proyectos web?
GitHub es una plataforma en la nube para almacenar proyectos Git y trabajar en equipo.
